## Environments setup
- create 2 virtual environemnt with Python 3.11:
    - 'sinnven' which will be used to run the notebooks: test_case_1_sinn.ipynb; test_case_2_sinn.ipynb; test_case_1_unet.ipynb; test_case_1_fno.ipynb
```
python3.11 -m venv .sinnvenv
source .sinnvenv/bin/activate
pip install -r requirements_sinn.txt
```
    - 'gnnven' which will be used to run the notebook: test_case_1_gnn.ipynb
```
python3.11 -m venv .gnnvenv
source .gnnvenv/bin/activate
pip install -r requirements_gnn.txt
```
